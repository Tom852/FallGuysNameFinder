Changelog

v2.2

  • Hotfix to fit the UI patch (slightly different screenshot area) and the nameplate that is fixed to the blue zic-zac lines with crown by now.
  • If it stays like this and isn't a bug, flexibility for different nameplates is no longer necessary and specific screen-shots can be taken for this plate, but I'll wait implementing this.

v2.1

  • Support any resolution narrower than 16:9

v2.0

  • Added pool matching
  • Added count of names that match
  • Added preview output of names that match
  • Added global error handler
  • Added always-on-top option for single monitor users
  • Improved handling of server timeouts
  • Taking fullscreen screenshots on major errors for better debugging, especially on server disconnects

v1.0

  • Added support for 21:9 aspect ratio
  • Added option to not press ESC on success
  • Improved user guide
  • Improved log output
  • Implemented many small user feedbacks
  • Prepared support for pool matching
  • Improved internal name capitalization and strict internal usage of 'StringTriple', 'Pattern' and 'Name'
  • Reduce time between rerolls
  • Tesseract word dawg renewed
  • Removed word options that were not recorded during testing.

v0.6

  • Added chance to hit and time estimate
  • Stats are now alphabetical
  • Press ESC at end

v0.5

  • Added scoreboard for fuzzy matches instead of taking max-value
  • Added changelog
  • About section as html

v0.4

  • Added statistics tracking
  • Detect if name gets parsed successful, but does not change. If so, press SPACE to eliminate a possible Amazon-Link window or similar.
  • Increase name parsability when overlay window removes contrast from nameplate.
  • Massive refactoring

v0.3

  • Instead just one raw-text refinement, use three: soft artifact remover, hard artifact remover, word detector
  • Use approximate matching on parsing failure
  • Add support for windowed mode
  • Patterns can now be added by dropdown list

v0.2

  • UI design
  • Added about section
  • Added user guide section
  • UI now displays Fall Guys window status and engine status
